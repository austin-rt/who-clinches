## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript (strict mode)
- **Styling**: Tailwind CSS + DaisyUI components
- **State Management**: Redux Toolkit (RTK) + RTK Query
- **Icons**: React Icons
